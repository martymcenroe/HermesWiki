# HermesWiki

Documentation for **Hermes** — an autonomous email agent that triages inbound recruiter outreach, built on Cloudflare Workers and AWS Bedrock (Claude).

**[Read the Wiki →](https://github.com/martymcenroe/HermesWiki/wiki)**

The wiki covers the architecture, conversation state machine, guardrails and evaluation, and the security and cost engineering behind the system — including threat modeling against the OWASP LLM/Agentic Top 10 and MITRE ATT&CK/ATLAS, NIST alignment, and a full cost breakdown.
